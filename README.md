```
10s @ 60fps = 600 frames

Per-Frame Change = (End - Start) / Total Frames
Example: Complexity 50‚Üí200 over 240 frames
(200 - 50) / 240 = 0.625 per frame

Check halfway: Frame 120 should show ‚âà 125
Scrub to frame 120 to verify!
```
